== README

Simple 1 page app to import data from csv into PostgreSQL database and display imported data in a dual line graph using Highcharts.

How to launch the app:

* Ensure postgresql is installed on your computer

* <tt>Bundle Install</tt>

* <tt>Rake db:create</tt>

* <tt>Rake db:migrate</tt>

* <tt>Rake data:import</tt>

* <tt>rails s</tt>
